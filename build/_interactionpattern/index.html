ytplayer

YouTubeの動画埋め込み

（例）
<%- include("/common/components/ytplayer/_ytplayer.ejs", {
  youtubeID: "ynccI1WB300",
  width: "1000",
  height: "564",
  autoplay: "true",
  controls: "false",
  fs: "false",
  loop: "true",
  rel: "false",
  modifiers: []
}) %>

↓ build

<div class="ytPlayerArea ytPlayerArea--noTouch js-yt-player" data-yt-id="ynccI1WB300" data-yt-width="1000" data-yt-height="564" data-yt-autoplay="true" data-yt-controls="false" data-yt-fs="false" data-yt-loop="true" data-yt-rel="false">
  <div class="ytPlayerFrame js-yt-frame"></div>
</div>

主なパラメータは以下になります。
autoplay ... 自動再生のON/OFF。
controls ... 動画プレイヤーのコントローラーの表示/非表示。
fs ... 全画面表示の可否。
loop ... 動画が終了した際のループ再生のON/OFF。
rel ... 動画終了後の関連動画を「同じチャンネルのものにするか」の選択ができます。
